# seed-import

과거 검수 완료 자막 = 우로보로스 루프의 초기 연료.

## 절차

1. 인자 또는 사용자에게 .srt 파일들이 있는 폴더 경로 확인. `data/seeds/`로 복사 권장.
2. `uv run jamak seed-import <폴더>` 실행.
3. 결과 보고: 파일 수, 추출된 용어 후보 수.
4. **중요**: 자동 추출 용어는 `approved=False` 상태 — 프롬프트에 아직 반영 안 됨.
   임포트 직후 `/glossary-review`를 이어서 실행해 진짜 고유어휘(신인, 축지법, 하늘궁,
   불교/유교/기독교 용어 등)를 승인하도록 안내하거나 바로 진행.

## 주의

- 인코딩: UTF-8 우선, 실패 시 CP949 폴백 (코드에 내장).
- 같은 폴더 재임포트는 안전 (기존 용어 스킵).
