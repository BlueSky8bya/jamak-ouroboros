# PLAN-20260710-004 — Linked Timing Controls

## Goal

타이밍 버튼의 개념 부담을 줄이고, 현재 시간으로 시작/끝을 맞출 때 이웃 자막까지 자동으로 맞물리게 한다.

## Risk Level

L1/L2 — 검수 웹앱 UX와 세그먼트 타이밍 API 변경. DB 스키마와 학습 데이터는 변경하지 않는다.

## Scope

- `시작/끝/경계/나눔` 버튼을 더 명확한 행동 버튼으로 축소.
- 현재 자막 시작을 현재 시간에 맞출 때 이전 자막 끝도 같이 조정.
- 현재 자막을 현재 시간에서 다음 자막으로 넘길 때 현재 끝/다음 시작을 같이 조정.
- 수동 시간 입력도 가능한 범위에서 이웃 자막과 맞물리도록 조정.

## Verification

- Python 문법 검사.
- 임시 SQLite로 previous/next linked timing 스팟체크.
- 프론트엔드 빌드.

## Outcome

- Replaced four visible timing buttons with two action-oriented buttons:
  - `여기서 시작`
  - `여기서 넘김` / `여기서 끝`
- Added previous-boundary API so the current row start and previous row end move together.
- Updated next-boundary API so the last segment can still end at the current time.
- Manual start/end edits now prevent overlaps by adjusting the adjacent segment only when needed.
- Removed visible `나눔` control from rows to reduce conceptual load. The backend helper remains available but is no longer shown as a primary action.
