# PLAN-20260710-002 — Current-Video Feedback Propagation

## Goal

`고친 내용 학습시키기`가 다음 영상만 좋아지게 하는 버튼처럼 보이지 않도록, 현재 검수 완료된 자막에서 배운 교정쌍을 같은 영상의 뒤쪽 미검수 자막에 즉시 반영한다. 기본 경로는 API를 쓰지 않는 결정적 치환으로 유지한다.

## Risk Level

L2 — 피드백 루프와 정확도 체감에 영향을 준다. DB 스키마는 변경하지 않는다.

## Scope

- `src/jamak/feedback.py`: 검수 완료 세그먼트의 위치를 기준으로 뒤쪽 미검수 세그먼트에 학습쌍 전파.
- `src/jamak/web/app.py`: export 시 자동 흡수 결과가 실제 내보내기 데이터에 반영되도록 순서 조정.
- `src/jamak/web/frontend/src/*`: 학습 버튼 전 저장 완료 보장, 현재 화면 재조회, 반영 통계 표시.
- `docs/agent/*`: 상태/변경 로그 갱신.

## Non-Goals

- Claude 재호출 기반 전체 재교정.
- DB 스키마 변경.
- 학습 데이터를 코드나 프롬프트에 하드코딩.

## Verification

- `uv run jamak doctor`
- 임시 SQLite DB를 사용한 피드백 전파 스팟체크.
- 가능하면 프론트엔드 빌드 또는 타입체크.

## Outcome

- Implemented scoped current-video propagation from reviewed correction pairs to later unreviewed subtitles.
- Kept propagation deterministic and zero-API; global high-confidence pairs still apply to any unreviewed segment.
- Fixed the review UI race by awaiting pending saves before absorb/export/back navigation.
- Export now absorbs first and then reads segments so propagated drafts are included in the downloaded SRT.
