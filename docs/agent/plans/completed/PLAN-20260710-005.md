# PLAN-20260710-005 — Frictionless Delete And Undo

## Goal

삭제 확인창을 제거하고, 실수해도 `Ctrl+Z`로 세그먼트 상태를 복구할 수 있게 한다. 작업창도 더 명확하고 검수 친화적으로 다듬는다.

## Risk Level

L1/L2 — 검수 웹앱 UX와 세그먼트 복원 API 추가. 학습 DB 스키마는 변경하지 않는다.

## Scope

- 삭제 버튼 즉시 실행.
- split/merge/delete/timing 조작 직전 세그먼트 스냅샷 저장.
- `Ctrl+Z` 및 버튼으로 마지막 세그먼트 조작 복구.
- 왼쪽 작업 패널과 단축키/상태 표시 polish.

## Verification

- Python 문법 검사.
- 임시 SQLite로 delete -> restore 스팟체크.
- 프론트엔드 빌드.

- Removed delete confirmation from subtitle rows.
