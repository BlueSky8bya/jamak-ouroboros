# PLAN-20260710-006 — Continue Workflow And Delete Shortcut

## Goal

Let the reviewer continue without choosing a filter view, and let them delete the current subtitle cell quickly with `Delete` while preserving `Ctrl+Z` recovery.

## Risk Level

L1 — review webapp UX only. No DB schema or learning-data source changes.

## Completed Scope

- Removed the `먼저 볼 곳` / `안 본 곳` filter UI and the `Filter` type.
- Added `이어서 작업하기`, which flushes pending edits and jumps to the next unreviewed subtitle from the focused/current position.
- Changed Enter-confirm navigation to follow the same next-work target.
- Added `Delete` shortcut for the focused/current subtitle when the user is not typing.
- Kept deletion on the existing segment undo stack so `Ctrl+Z` restores the cell.
