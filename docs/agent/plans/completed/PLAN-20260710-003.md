# PLAN-20260710-003 — Review Timing Ergonomics

## Goal

검수자가 현재 편집 중인 자막과 영상 재생 위치를 헷갈리지 않게 하고, 자막을 합치거나 나눈 뒤 시간 경계를 한 줄씩 수동 보정하는 피로를 줄인다.

## Risk Level

L1/L2 — 검수 웹앱 UX와 세그먼트 시간 편집 API 변경. DB 스키마와 학습 데이터는 건드리지 않는다.

## Scope

- 현재 재생 자막과 편집 중 자막을 UI에서 별도 표시.
- 플레이헤드 기반으로 현재 줄 시작/끝 및 다음 줄 경계를 한 번에 맞추는 무API 버튼 추가.
- 현재 줄과 다음 줄의 총 구간을 텍스트 길이 비율로 재분배하는 무API 버튼 추가.
- 합치기 시 인접 자막의 반복 접두/접미 텍스트를 자동 제거.

## Non-Goals

- LLM/API 기반 재타이밍.
- DB 스키마 변경이나 word timestamp 영구 저장.
- 전체 영상 강제 재분할.

## Verification

- Python 문법 검사.
- 임시 SQLite로 merge/reflow/boundary 스팟체크.
- 프론트엔드 빌드.

## Outcome

- Added separate playback/editing status in the left panel.
- Added a compact timing strip around the focused or current playback area.
- Added per-row playback rail when the video is inside that segment.
- Added zero-API timing actions:
  - current time -> segment start
  - current time -> segment end
  - current time -> boundary between current and next segment
  - redistribute current+next span by text length
- Made merge remove repeated adjacent overlap by default.
