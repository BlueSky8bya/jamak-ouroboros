# PLAN-20260710-009 - Pronoun-Safe Feedback Propagation

## Goal

Prevent the feedback absorption loop from learning or applying pronoun/reference rewrites such as `그 여자` -> a proper name. Subtitle text should follow what is spoken, so pronouns and demonstrative references must remain pronouns unless the user manually edits that specific cell.

## Risk Level

L2 - learning/propagation behavior. No DB schema change or destructive seed-data mutation.

## Scope

- Add a generic guard for Korean pronoun/demonstrative reference phrases.
- Skip unsafe pairs during pair extraction, deterministic propagation, global pre-pass, and few-shot prompt construction.
- Keep ordinary misrecognition fixes such as `수로보관` -> `수로보가네` eligible.
- Repair already over-propagated unreviewed machine suggestions when restoring moves text closer to the STT source.

## Result

Done. Added `src/jamak/learned_pairs.py` and wired it into feedback absorption, same-video propagation, global pre-pass, and few-shot correction loading. The LLM correction prompt now explicitly preserves pronouns/demonstratives, and correction cache prompt version moved to `v3`.
