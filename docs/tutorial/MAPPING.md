# 코스 × 대본 × 단축키 매핑

투어 단계(실제 동작을 해야 진행) ↔ 그 동작을 유도하는 대본 줄 ↔ 단축키/기능.
리허설 때 이 표로 "대본이 정말 그 조작을 유도하는지" 점검한다.

## 코스 1 · 기본기 (연습1-기본기.md)

| 투어 단계 (advance 이벤트) | 유도 대본 줄 | 단축키/기능 |
|---|---|---|
| ▶ 재생 클릭 (`play`) | #1~3 "귀로 들어 보세요" | ▶ 재생 |
| 확인 (`confirm`) | #4~6 쉬운 문장 + "맞으면 엔터" | Enter |
| 자막 글 클릭 (`open-row`) | #7~10 오인식 미끼(깻잎·밤나무·축지법·공중부양) | 클릭→고치기→Enter |
| 🙉 (`hold`) | #11 웅얼 문장 + #12 안내 | Alt+H / 🙉 |
| 되돌리기 (`undo`) | #13 "알트 제트" | Alt+Z / ↶ |
