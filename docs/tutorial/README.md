# 튜토리얼 연습 영상 (코스 1~6 전용)

따라하기(🎓) 코스마다 전용 연습 영상을 만든다. 영상의 **내레이션 자체가 교재**이고,
각 코스에서 익힐 조작이 **진짜 필요해지도록 연습 재료를 일부러 심는다**
(빠른 말 → 되감기/배속, 긴 침묵 → 건너뛰기, 긴 문장 → 나누기, 반복 오인식 단어 →
찾기·바꾸기, 웅얼거림 → 🙉 …).

## 파이프라인

1. `scripts/연습N-*.md` 대본 (이 폴더) — 사람이 검토하는 문서이자 렌더 입력
2. TTS: edge-tts (한국어, 무료) — 대사별 mp3 생성, `스타일` 열이 rate/volume 결정
3. 합성: Remotion(React) — **발화 문장은 화면에 표시하지 않는다**(듣기 검수
   연습이 목적 — PLAN.md §2.5가 이 문서를 supersede). 화면은 코스 배지 +
   진행 점 + 파형 펄스 + 스타일 아이콘. 오디오는 PCM으로 합쳐 단일
   audio.wav (`뒤 쉼` 열이 대사 사이 침묵 길이, 타이밍은 샘플 오프셋 실측)
4. 렌더: `npx remotion render` → mp4 6개 (1080p, 1~2분/개)
5. 유튜브 업로드(미등록) → 앱에 링크 등록 → 워커 처리 → 🎓 연습용 지정
6. 코스↔연습영상 연결 → 코스 시작 시 해당 영상 자동 열림

## 대본 표 형식

| 열 | 의미 |
|---|---|
| 대사 | TTS가 읽을 문장 (그대로 음성이 됨) |
| 스타일 | `보통` · `빠르게`(rate +40%) · `느리게`(rate -20%) · `웅얼`(rate +30%, volume 35% — 🙉 연습용) · `침묵`(대사 없음, 쉼만) |
| 뒤 쉼(초) | 이 대사가 끝난 뒤 침묵 길이 |

## 주의

- **오인식은 확률적**이다. 대본은 STT가 흔들리기 쉬운 재료(고유명사 "뭉치",
  도메인 용어 "축지법·공중부양", 웅얼거림)를 넣었지만, 축지법·공중부양은
  **whisper hotwords에 이미 있어 맞게 인식될 확률이 높다** — 실제 처리 후
  화면 초안 기준으로 판정하는 리허설(PLAN.md §5 predicate 표, 최대 2회)을
  반드시 거칠 것.
- 연습 영상은 반드시 **🎓 연습용 지정** — absorb·줄길이 학습·번역 few-shot·
  학습 export·CER 평가 전부에서 제외되어 연습 편집이 학습 데이터를 오염하지
  않는다 (CHG-20260714-005).
- 내레이션 속도 기본값: 또박또박 (rate -5%), 여성/남성 목소리는 렌더 시 결정.
