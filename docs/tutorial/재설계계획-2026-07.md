# 연습 영상 재설계 계획 (2026-07-16)

## 원칙 (정책 2026-07-16)
- 연습 영상은 **무-API 도구만** 가르친다. API 쓰는 것(맞춤법 검사·번역 생성·초안)은 관리자 전용이라 **아예 안 가르침**.
- 실제 검수에서 **숨긴 도구**(안심 확인·복구 채우기)도 안 가르침 — 배워도 실전에서 못 씀.
- 스텝 내용을 바꾸려면 **영상 재렌더 필수**(나레이션이 교재라 코드만 바꾸면 어긋남).

## 6개 코스 — 무엇을 가리킬지

### 연습1 · 기본기 (basic) — **변경 없음, 재렌더 X**
영상 재생 → 위에서부터 Enter 확인 → 틀린 글자 고치고 Enter → 🙉 보류 → 되돌리기(Alt+Z) → 큰 글씨.
→ 전부 무-API. 그대로 둔다.

### 연습2 · 재생 다루기 (playback) — **변경 없음, 재렌더 X**
Tab 재생/멈춤 → 3초 뒤로 → 배속(느리게) → 3초 앞 → 10초 이동 → 이 자막 처음부터 → 구간 반복 → 자막 사이 이동.
→ 순수 키보드, 무-API. 그대로.

### 연습3 · 빠르게 훑기 (fast) — **변경 O, 재렌더 O**
- ❌ 삭제: **✅ 안심 확인** (실검수에서 숨김 — 배지가 오타 못 거름)
- ✅ 유지: 🔎 찾기·바꾸기(같은 오타 일괄) · '편집 시작 시 멈춤' 끄기 · '자동 따라가기' 설정 · 미확인만 순회
- → 리드를 **찾기·바꾸기**로 시작하도록 재구성.

### 연습4 · 나누기·합치기 (structure) — **변경 없음, 재렌더 X**
자막 글 선택 → 나누기 → 합치기 → 되돌리기.
→ 무-API. 그대로.

### 연습5 · 타이밍 (timing) — **변경 O, 재렌더 O + 신규 UI**
현재 문제: 타임라인 손잡이(`.timing-strip`)는 영상 밑(왼쪽)에 있으나, 타이밍 편집
버튼(Alt+[ ]·발화맞춤 `.timing-tools`)은 **오른쪽 자막 셀 안**에 있어 시선이 좌↔우로 튄다.
- ➕ **신규 UI (구현 필요)**: 영상 바로 밑(타임라인 옆/아래)에 **타이밍 편집 바** 신설 —
  지금 편집 중인 자막의 시작(Alt+[)·끝(Alt+])·발화 맞춤(Alt+\)을 **영상 밑에서 바로**.
  영상 재생바로 그 지점 미리보기 → 밑에서 손잡이 끌거나 Alt 단축키 → 다음 자막.
  오른쪽 셀 안 버튼은 유지(둘 다 동작)하되, **튜토리얼은 영상 밑 흐름을 가르친다**.
- 나머지 유지: ② 타이밍 탭 → ✨ 자동 정리 → 문제만 순회 → ✂ 무음 다듬기 → ⏱ 타이밍 검수 완료.
- → 전부 무-API. **신규 편집 바 구현 후** 재렌더.

### 연습6 · 마무리 (finish) — **전면 재설계: 실전 프로세스 캡스톤, 재렌더 O**
"버튼 하나씩"이 아니라 **실제 검수 한 흐름을 처음부터 끝까지 한 번 몸에 익히게** 한다.
편집 재료를 일부러 섞은 클론(일치 자막 여러 개 + 필요없는 대답 "음·어" 셀 + 합칠 조각 +
길어서 나눌 문장 + 한자어 + 동음이의 함정)에서 아래 흐름을 순서대로:

1. **속도 설정 판단** — 자신 있으면 `편집 시작 시 멈춤 끄기` + `자동 따라가기 끄기`,
   자신 없으면 반대로 켜기. (나레이션이 "본인 속도에 맞게 고르세요" 유도)
2. **일치 자막은 Enter 촤르륵** — 맞는 자막은 Enter로 빠르게 확인하며 내려감
   (편집 시작 시 멈춤 꺼야 안 끊김).
3. **필요없는 대답 셀 삭제** — "음..", "어" 만 있는 셀은 **Alt+Delete**로 즉시 삭제.
4. **합치기** — 위아래로 쪼개진 걸 **Ctrl+Shift+Enter**로 합침.
5. **나누기** — 한 셀에 두 문장이면 커서 옮겨 **Ctrl+Enter**로 나누고 **Enter 한 번 더**
   (앞 셀 확인 완료).
6. **다시 보기** — 놓친 것 같으면 **Alt+↑/↓**로 셀 오르내리며 재확인,
   **뒤로가기(3초)·재생/일시정지 단축키**로 그 부분 다시 듣기.
7. **漢 한자 채우기** — 漢 눌러 미리보기 → 각 행 **▶ 시각**으로 영상 재생 →
   흑판 한자인지/동음이의 맞는지 들으며 판단(함정 "보시는"의 보시 ≠ 布施 → 체크 해제) → 적용.
8. **📚 학습(Alt+K)** → **자막 받기(점검표)**로 마무리.

- ❌ 삭제: 🛠 복구·채우기 · 마지막 "✏️ 맞춤법 검사" 언급(관리자 전용).
- 💬 미리보기(Alt+P)는 흐름 중 "최종 확인" 단계로 녹임.
- 결함 주입 재설계 필요: `practice.py` finish 클론에 위 6종 재료를 결정적으로 심기.
