2. **실검수에 안 보이는 버튼은 안 가르친다.**
3. 대사 바꾸면 **재렌더 필수** (나레이션이 교재).

## 0.5 크기 제약 (2026-07-16 발견, v0.9.52에서 1차 해소)

연습 영상은 에디터 **왼쪽 열 안**에서 재생된다: `.left = minmax(390px, 440px)`, padding 14
→ 실제 표시 폭 **~362px**. 캔버스 1920 대비 **배율 0.19**.

| 캔버스 | 화면 | 판정 |
|---|---|---|
| 40px (구버전 본문) | 7.6px | ❌ 판독 불가 |
| 16px 진행 점 ×17개 | 3px | ❌ 안 보임 |
| **84px** | 16px | ✅ 최소선 |
| **88~92px** (현행 키캡·버튼) | 17px | ✅ |

**규칙**: 읽어야 할 글자는 **캔버스 84px 이상**. 얇은 선·작은 점·촘촘한 와이어프레임은
저배율에서 회색 뭉개짐 → 굵고 성기게. 새 시각 큐 추가 시 **362px로 축소해 눈으로 확인**
후 확정한다(`out/check/*_zoom.png` 절차).

부작용: 캔버스에 들어가는 정보량이 줄었다 → 한 대사에 시각 블록 **2개까지**.
그 이상 필요하면 대사를 쪼갠다.

## 2. 연습1 · 기본기 — **유지**

▶ 재생 → 위에서부터 Enter 확인 → 오인식 자막 눌러 고치고 Enter → 🙉 잘 안 들림 → Alt+Z → 글씨 크게.
전부 무-API·실검수 노출 기능. **손대지 않는다.**

## 3. 연습2 · 재생 다루기 — **유지**

Tab → Shift+Tab(3초 뒤) → 0.75× → Ctrl+→ → Ctrl+Shift+←→(10초) → Ctrl+\(구간 처음) → 🔁(Alt+R) → Alt+↑↓.
순수 키보드. **손대지 않는다.** 연습6 캡스톤이 이 단축키들을 실전 맥락으로 되살린다.

## 6. 연습5 · 타이밍 — **재설계 (영상 밑 편집 바 중심)**

**현재 문제**(사용자 지적): 타임라인 손잡이는 **영상 밑 왼쪽**, 타이밍 편집 버튼(Alt+[ ] \·발화맞춤 `.timing-tools`)은
**오른쪽 자막 셀 안** → 시선이 좌↔우로 튄다. 미리보기는 영상 재생바에서 하는데 수정은 저 멀리서.

### 🆕 선행 코드 A — "영상 밑 타이밍 편집 바" (`.timing-bar`)

- 위치: 영상 바로 아래, `.timing-strip`(타임라인) 옆/아래 한 줄.
- 내용: **지금 편집 중인 자막**의 `[여기서 시작 Alt+[] [여기서 넘김 Alt+]] [⤢ 발화 맞춤 Alt+\]` + 시작/끝 시각 표시.
- 흐름: 영상 재생바로 그 지점 미리보기 → **눈이 영상에 붙은 채로** 바로 밑에서 손잡이 끌거나 Alt 키 → Alt+↓ 다음 자막.
- 오른쪽 셀 안 `.timing-tools`는 **유지**(둘 다 동작). 튜토리얼만 영상 밑 흐름을 가르친다.
- ② 타이밍 모드에서만 노출. 편집 중 자막 없으면 비활성 안내.
- **실검수에도 바로 이득** — 튜토리얼과 별개로 값어치 있는 실기능 개선.

### 새 스텝 순서

| # | target | 동작 |
|---|---|---|
| 1 | `.mode-tabs` | ② 타이밍 탭 |
| 2 | `.issue-bar` | 다음 문제 → (기계가 아니라 **사람이** 문제만 골라 봄) |
| 3 | `.timing-bar` 🆕 | **영상 밑 바**에서 Alt+[ 여기서 시작 |
| 4 | `.timing-bar` 🆕 | Alt+] 여기서 넘김 |
| 5 | `.timing-bar` 🆕 | Alt+\ ⤢ 발화 맞춤 |
| 6 | `.timing-strip` | 손잡이 드래그 미세조정 |
| 7 | `.tool-tighten` | ✂ 무음 다듬기 (Alt+M) |
| 8 | final | 순서 정리 + ⏱ 타이밍 검수 완료 |

- ❌ 삭제: **✨ 타이밍 자동 정리** 스텝 (ADR-0012로 폐지 — 실검수에 없는 버튼).
- 대사: "오른쪽 자막 아래 시간 도구" → **"영상 바로 밑 시간 줄"**로 위치 안내 전면 교체.
- 대사 3번("타이밍 자동 정리 버튼을 누르면 기계가 한 번에 맞춰 줍니다")과 12번
  ("자동 정리 먼저, 남은 문제만 손보기") **삭제/교체** — 새 흐름은 **"문제 큐로 골라
  → 영상 밑 바에서 직접"**. 기계가 먼저 하는 단계가 사라졌으므로 코스의 리드가 바뀐다.
- ⚠️ **재렌더 전까지** `.tool-auto`는 practice 모드에만 남겨 둔다(나레이션이 아직
  지시하므로 투어가 안 깨진다). **재렌더 후 완전 삭제** — `retime.py`·`/auto-timing`
  엔드포인트까지 함께.
