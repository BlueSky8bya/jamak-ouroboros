# 연습 영상 6 — 마무리: 실전 프로세스 캡스톤

- 코스: 6️⃣ 마무리 (속도설정 → Enter 촤르륵 → Alt+Delete → 합치기/나누기 → 되짚기 → 漢 한자 채우기 → 💬 → 📚 → 자막 받기)
- 목표 길이: 약 165초
- 재설계(2026-07-17): "버튼 하나씩" 나열식 폐기 → **실제 검수 한 흐름을 처음부터 끝까지 한 번**.
  ❌ 삭제: 🛠 복구·채우기(실검수에서 숨김) · ✏️ 맞춤법 검사 언급(관리자 전용).
- 심은 재료 — **나레이션이 곧 드릴 재료다** (이 영상의 자막 = 이 나레이션의 STT):

  | 재료 | 어디서 나오나 | 드릴 |
  |---|---|---|
  | ① 정확한 자막 4개 연속 | #4~#7 (짧고 쉬운 문장) | Enter 촤르륵 |
  | ② 필요없는 대답 셀 2개 | #8 "음...", #9 "어." (앞뒤 쉼으로 별도 셀 보장) | Alt+Delete |
  | ③ 합칠 조각 1쌍 | #11+#12 (한 문장을 중간 쉼으로 끊어 말함) | Ctrl+Shift+Enter |
  | ④ 두 문장 든 긴 셀 | #14 (쉼 없이 두 문장 연달아) | Ctrl+Enter → Enter |
  | ⑤ 흑판 한자어 | #17 "해탈", "열반" (사전 tier=special) | 漢 채우기 → 적용 |
  | ⑥ 동음이의 함정 | #18 "영상을 **보시는** 분들" (보시=布施도 special이라 오채움) | 모달에서 **안 바꿈** |

- 주의: ②③④는 STT 분절에 의존한다 → 렌더 후 실제 분절을 확인하고, 어긋나면
  `practice.py`의 finish 결함 주입으로 결정적으로 보정한다 (재설계계획 §7-B).
