# 대본 수집함 (corpus inbox)

다른 분들이 작성한 강연 대본 `.txt` 파일을 **이 폴더에 그대로 넣으세요**.
하위 폴더로 정리해도 됩니다 (예: `data/corpus/오늘의신인님/`, `data/corpus/카페대본/`).

여기 든 파일로 하는 일:

1. **한자어 채굴** — `말(漢字)` 병기 패턴과 "뜻 음 자" 풀이를 찾아
   `HanjaTerm` 사전(흑판 특수어)에 추가
2. **특화 단어 추출** — 신인님 강연에만 나오는 표현을 용어사전(glossary)
   후보로 추출해 STT 교정에 반영

처리는 에이전트(Claude)에게 "corpus 폴더 채굴해줘"라고 요청하면 됩니다.
새로 추가된 항목은 기존 사전과 충돌 검사(동음이의 등) 후 DB에 들어가고,
`docs/hanja-lexicon.md`가 재생성됩니다.

## 파일명 규칙 (2026-07-16 통일)

```
[시리즈] NNNN회 YYYY.MM.DD 제목.txt   ← 회차 시리즈 (토요강연·일요강연·하늘궁)
[시리즈] YYYY.MM.DD 제목.txt          ← 날짜 시리즈 (오늘의신인님·본좌이슈·전화인터뷰·허팅·콜미본좌)
[정리] 제목.txt                       ← 회차 없는 용어·주제 글
[미상] YYYY.MM 제목.txt               ← 시리즈 판별 불가
```

- 회차는 0채움(토요 4자리, 일요·하늘궁 3자리) — 정렬용
- 개명 이력은 `_rename_log.tsv` (원래 이름 ↔ 새 이름, 복구 가능)
- 최선우님 게시글 추출 결과: `_최선우-다국어-용어정리.md`

## 규칙

- 인코딩: UTF-8 권장 (메모장 "다른 이름으로 저장" 시 인코딩 UTF-8 선택).
  EUC-KR/CP949도 읽을 수는 있지만 UTF-8이 안전합니다.
- 파일명에 출처를 남기면 좋습니다 (예: `836회_불교강연_홍길동정리.txt`).
- **이 폴더의 .txt는 git에 올라가지 않습니다** (공개 저장소이므로 대본
  원문 보호 — .gitignore 처리, 이 README만 추적됨).
- 원본은 지우지 마세요. 채굴은 읽기만 하며, 같은 파일을 다시 채굴해도
  중복 등록되지 않습니다.
